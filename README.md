# Game-TestCode

This is a small project built in Unity to experiment with player movement, animation, and camera system.  
It is part of my personal game development

- Player Movement: 2D movement system
- Animations: Basic animation controller for player actions  
- Camera System: Follow camera with adjustable offsets and height  

- Currently in early development  
- Core systems implemented (movement, animation, camera)  
- No enemies, combat, or full gameplay yet

Technologies Used
- Unity  
- C#
